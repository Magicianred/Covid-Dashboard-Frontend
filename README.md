# Covid-19 Dashboard

Covid-19 Dashboard is a free app to display helpful information and news regarding Covid-19.

Created using React, Python, and PostgreSQL

Link to [Travel Buddy](https://choosealicense.com/licenses/mit/)

## Installation

```
npm install
npm start
```
